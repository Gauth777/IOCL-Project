# IOCL News Dashboard

A real-time news monitoring and analytics platform for IndianOil Corporation Limited (IOCL) with a Node.js/Express backend and a dynamic frontend.

## Project Structure

```text
IOCL-Project/
├── package.json
├── server.js
├── data/
│   └── news.json
└── public/
    ├── index.html
    ├── style.css
    ├── script.js
    └── assets/
        └── images/
```
